# Scelta Infinity Website

Production-ready Next.js 14+ showcase website for **Scelta Infinity**.

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style reusable components

## Setup
```bash
npm install
npm run dev
```

## Notes
- Pure showcase website (no checkout/cart/payment).
- Book purchase buttons redirect externally to Amazon.
- Affiliate links redirect to third-party platforms.
- Contact form uses Formspree placeholder endpoint.
